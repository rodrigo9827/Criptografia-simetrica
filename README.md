# Criptografia-simetrica


* Cifra de Cesar: 
    
    * arquivo: criptografia_cifra_cesar
    o programa pede uma menssagem para ser criptografada, logo após solicita a chave de deslocamento que é o equivalente a quantidade de casas que a letra terá de andar para ser criptografada
        
       * exemplo de uso: Digite a menssagem a ser criptografada: (Texto simples sem assento, case_sensetive) 

       Qual a chave de deslocamento: (Valor numérico inteiro ...-2,-1,0,1,2...)
    * arquivo: descriptografia_cifra_cesar 
    o programa pede uma menssagem para ser descriptografada, logo após solicita a chave de deslocamento que é o equivalente a quantidade de casas que a letra terá de andar para ser descriptografada
        * exemplo de uso: Digite a menssagem a ser descriptografada: (Texto simples sem assento, case_sensetive) 

       Qual a chave de deslocamento: (Valor numérico inteiro ...-2,-1,0,1,2...)
