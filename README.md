# Criptografia-simetrica

* Cifra de Cesar: 
    
    * arquivo: criptografia_cifra_cesar
    o programa pede uma menssagem para ser codificada, logo após solicita a chave de deslocamento que é o equivalente a quantidade de casas que a letra terá de andar para ser criptografada
        
       * exemplo: Digite a menssagem a ser criptografada: (Texto simples sem assento, case_sensetive) 

       Qual a chave de deslocamento: (Valor numérico inteiro ...-2,-1,0,1,2...)
